## SEENHUB 📦

**나만의 취향을 기록하는 특별한 공간**

SEENHUB는 음악, 영화, 도서 등 좋아하는 콘텐츠를 기록하고 감상할 수 있는 개인 아카이브입니다. <br />
자신의 취향을 자유롭게 탐색하고, 인상 깊었던 순간들을 조용히 저장할 수 있는 공간입니다. <br />

### 🔗️ Go to Service
